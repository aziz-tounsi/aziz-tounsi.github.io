# Assets Directory

This directory contains images for the portfolio website.

## Required Images

Please add the following images to this folder:

1. **profile.png** - Profile avatar image (3D avatar or professional photo)
   - Recommended size: 140x140px or larger (will be cropped to circle)
   - Format: PNG with transparency preferred

2. **sage-stone-preview.png** - Screenshot of the SAGE & STONE e-commerce website
   - Source: https://site-vitrine-khaki.vercel.app
   - Recommended size: 400x200px or similar aspect ratio

3. **educap-preview.png** - Screenshot of the Educap AI Assistant
   - Source: https://educap-assistant-production-7178.up.railway.app
   - Recommended size: 400x200px or similar aspect ratio

## How to Take Screenshots

1. Visit the live project URL
2. Use browser developer tools (F12) to set a specific viewport if needed
3. Take a screenshot of the main view/landing page
4. Save as PNG format
